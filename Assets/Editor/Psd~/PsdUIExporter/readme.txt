    记录本工具的规范和使用说明

一、图层命名与参数格式
    图层命名方式为：layer_name|param1&param2&...
	1、layer_name为图层名字，param1为参数，如没有参数，则只需要写图层名字即可
	2、使用|符号将图层名字和参数分隔
	3、可以带多个参数，每个参数使用&符号分隔
	4、图层名字不可带有特殊符号，如"|"、"&"
二、参数说明
	1、alpha 透明度标识符，包括该参数，则需要设置图层的透明度。本工具会获取图层的“填充”值，并将该值转换到Unity Image组件的Color.a值上。
	   例如：c_close1|alpha
	2、button 按钮组件标识符，包括该参数，则该图层生成预设时需要添加Button组件。可以在用于组或者图层上,
	   如果用于图层上，再按钮的底图就是该图层;如果用于组上，那么该组的最后一个子图层就是该按钮的底图。
	   例如:
	   c_close1|alpha&button
	   sure|button
	3、noimage 图片不导出，最终使用的是以动态加载方式加载图片，不使用psd里的样图。例如一些物品icon可以使用这个标签，因为物品icon单独导入
	4、mirror 图片为镜像结点，把localScale.x = -1
